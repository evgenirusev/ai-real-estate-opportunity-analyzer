# PROMPT INPUT 4 — Financial & Deal Structure Analysis

> **Purpose:** Captures the financial parameters specific to this deal — acquisition costs, financing, projected cash flows, and exit scenarios. Some fields are calculated, others are inputs.

---

## Instructions

Complete this for each specific property under evaluation. Use data from Inputs 1-3 to fill calculated fields.

---
