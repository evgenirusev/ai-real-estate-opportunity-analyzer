# Quick Screening Matrix (Pre-Assessment Filter)

> **Purpose:** A fast-lookup decision aid for screening properties before running the full assessment.

---

## Instructions

Use this matrix to quickly screen properties **BEFORE** investing time in the full 5-input assessment. Any **FAIL** = do not proceed unless there's a compelling strategic reason.

---

## Screening Checklist

| # | Criterion | Minimum Threshold | Status |
|---|-----------|-------------------|--------|
| 1 | Within budget (all-in) | ≤ €300,000 | ☐ PASS / ☐ FAIL |
| 2 | Estimated gross yield | ≥ 5% | ☐ PASS / ☐ FAIL |
| 3 | Location population trend | Stable or growing | ☐ PASS / ☐ FAIL |
| 4 | Clear title | Yes | ☐ PASS / ☐ FAIL |
| 5 | No structural red flags | No major issues | ☐ PASS / ☐ FAIL |
| 6 | Renovation ≤ 30% of price | ≤ 30% | ☐ PASS / ☐ FAIL |
| 7 | Demonstrated rental demand | Vacancy < 15% | ☐ PASS / ☐ FAIL |
| 8 | Legal purchase feasible | No prohibitive blocks | ☐ PASS / ☐ FAIL |
| 9 | Not in flood / hazard zone | No | ☐ PASS / ☐ FAIL |
| 10 | Reasonable exit (DOM < 180 days) | < 180 days | ☐ PASS / ☐ FAIL |

---

## Scoring Guide

| Result | Action |
|--------|--------|
| **10/10** | ✅ Proceed to full assessment |
| **8-9/10** | ⚠️ Proceed with caution, note failed criteria |
| **6-7/10** | ⚠️ Borderline — only proceed if failed criteria are fixable |
| **< 6/10** | ❌ Do not proceed |
