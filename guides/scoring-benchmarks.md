# Scoring Calibration Benchmarks (Bulgaria Market Context)

> **Purpose:** Bulgaria-specific benchmark values to calibrate scores accurately, since global benchmarks don't apply well to an emerging EU market.

---

## Acceptable Thresholds for "BUY" Recommendation

| Metric | Threshold |
|--------|-----------|
| Minimum net yield | ≥ 4.5% |
| Minimum cash-on-cash return | ≥ 6% |
| Maximum payback period | ≤ 15 years |
| Maximum renovation as % of purchase | ≤ 30% |
| Minimum cap rate | ≥ 5% |
| Maximum vacancy assumption (long-term) | ≤ 10% |
| Maximum vacancy assumption (short-term/seasonal) | ≤ 35% |
| Minimum break-even occupancy | ≤ 55% |
| Minimum DSCR (if financed) | ≥ 1.25 |

## Transaction Cost Benchmarks

| Cost Type | Typical Range |
|-----------|---------------|
| Property transfer tax | ~3% (varies by municipality, 0.1-3%) |
| Notary fees | 0.1-1.5% (tiered) |
| Legal fees | €500 – €2,000 |
| Agent commission (buyer side) | 2-3% (negotiable) |
| **Total acquisition overhead** | **~5-7% of purchase price** |
